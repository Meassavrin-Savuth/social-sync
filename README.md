# SocialSync

## 📌 Overview
SocialSync is a full-stack social media management platform built with a Go (Golang) backend and a Next.js frontend.

It allows users to manage multiple social media accounts, schedule and publish posts, upload and manage media, and view analytics in a single unified system.

The project is designed to simulate a real-world SaaS-style application with modern backend architecture and scalable system design.

---

## ✨ Features

- Multi-platform social media account management  
- Create, schedule, and publish posts  
- Automated background job processing for scheduled posts  
- Media upload and management using Cloudinary  
- Analytics tracking and reporting system  
- Role-based access control (RBAC)  
- Workspace-based organization structure  

---

## 🏗️ Architecture

### Backend (Go)
- Built using Go (net/http)
- RESTful API design
- Modular structure (routes, controllers, middleware, models)
- PostgreSQL database integration
- Background job processing (cron/schedulers)
- Authentication and authorization system

### Frontend (Next.js)
- Built with Next.js (App Router)
- Modern UI for managing posts, accounts, and analytics
- API integration with backend services

---

## 🛠️ Tech Stack

- Backend: Go (Golang)  
- Frontend: Next.js (React)  
- Database: PostgreSQL  
- Media Storage: Cloudinary  
- Authentication: JWT  
- Infrastructure: Docker, Docker Compose
